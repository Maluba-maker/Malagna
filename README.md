# Pocket Option Signal System (Streamlit)

## What this does
- Reads the live price from Pocket Option
- Generates BUY / SELL signals
- M1 logic
- Confidence %
- Reason
- Signal history
- No trading
- No automation

---

## How to deploy (Streamlit Cloud)

1. Upload:
   - app.py
   - requirements.txt

2. Go to https://streamlit.io/cloud
3. Deploy from GitHub
4. Done
